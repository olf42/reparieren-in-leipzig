Über

Diese Karte wurde von olf aus dem sublab im Rahmen von Code For Leipzig im Mai 2014 entwickelt. Die Daten werden von den Menschen von Café Kaputt aktuell gehalten.

Technische Umsetzung

Diese Website wurde mit Twitter Bootstrap und dem Theme Amelia mit Hilfe von Leaflet.js und JQuery umgesetzt.
Das Backend wurde in CherryPy implementiert. Die Datenbank wird mit SQLite verwaltet. Der Quelltext ist auf github zu finden.
Als Datenquelle wird Openstreemap benutzt, die Tiles von MapQuest bereitgestellt.
